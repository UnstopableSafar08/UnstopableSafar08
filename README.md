## About Me

I'm a DevOps Engineer with a strong focus on **infrastructure automation**, **production reliability**, and **scalable systems design**. My day-to-day spans Linux server administration, Kubernetes cluster management, CI/CD pipeline engineering, and building observability stacks that give teams real visibility into their systems.

I care deeply about clean automation, minimal toil, and keeping production stable — not just getting things to work, but making them *stay* working.

- Building and maintaining production Kubernetes environments (K3s, EKS)
- Designing observability pipelines with Grafana, Prometheus, Loki, and the ELK stack
- Automating deployments and configuration management with Ansible and Bash
- Managing high-throughput messaging systems with Apache Kafka (KRaft mode)
- Hardening and tuning Linux systems (RHEL/OracleLinux 9) for production workloads
- Cloud infrastructure on AWS with a focus on reliability and cost-efficiency
